## Analytics

GitHub Pages 배포 시 GA4 측정 ID를 `GOOGLE_ANALYTICS_MEASUREMENT_ID` Repository Variable로 설정하면 Google Analytics가 활성화됩니다.

현재 트래킹 이벤트:

- `nav_click`: 상단 브랜드/메뉴 클릭
- `cta_click`: 히어로 CTA 클릭
- `project_open`: 프로젝트별 서비스 열기 클릭
- `contact_click`: 푸터 이메일 클릭
